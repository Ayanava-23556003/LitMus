# LitMus
LitMus v1.0 is a 100% local, secure academic literature summarization tool powered by Ollama and `llama3.2`. Drop PDFs, DOCX, or text files to instantly distill deep research into a professional 500-word narrative and a structured 5-part matrix table (*Background, Objectives, Methods, Findings, Gaps*). Zero API keys, zero data leaks.
